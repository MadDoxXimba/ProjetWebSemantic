Projet Web Semantique - Année 2017 - 2018 - Informatique Décisionnelle

Ce projet a pour objectif d'analyser les données touristiques provenant de OpenData pour Nantes. Pour ce faire, on s'aidera de notions de bases RDF...

@author: Jonathan YUE CHUN

@author: Quentin LEVAVASSEUR

@author: Valentin BOUCHEVREAU

Le site étant héberger sur un serveur gratuit, il est normale que les requêtes soient lentes...

  • A noté, dans le cas où le scrollbox rechercher les offres par ville n'affichent rien, il faudra mettre le .ttl sur apache fuseki: https://herokufuseki.herokuapp.com/
  • Le fichier .ttl se trouve sur le répo: https://github.com/MadDoxXimba/apache_fuseki_heroku/tree/master/Data dans le dossier data
  • Dans le fichier main.py, on peut changer l'adresse du serveur en modifiant la variable: fusekiURL
